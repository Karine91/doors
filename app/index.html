Двери из ЭКОШПОНА

Экошпон – современный экологически чистый материал. Двери изготавливаются из волокон только натуральной древесины. На финальном этапе двери покрываются защитным слоем вещества искусственного происхождения, который обеспечивает долгий срок службы дверного полотна. Экологически чистое покрытие не выделяет вредных веществ.

Двери из СТЕКЛА

Цельностеклянные двери стали широко использоваться сравнительно недавно. Благодаря новым технологиям обработки стекла стало возможно их производство с усовершенствованными характеристиками. Теперь это прочный и безопасный материал для использования в дизайне интерьеров.
